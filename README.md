Лабораторная работа №3 - Параллельная обработка изображений
Инверсия цветов изображения с использованием шаблона Producer-Consumer.

Зависимости:
Скачать два заголовочных файла и положить рядом с main.cpp:

stb_image.h
stb_image_write.h

Поддерживаются форматы JPG и PNG. Файл должен лежать в той же папке, что и программа. Результат сохранится как photo_inverted.jpg.
